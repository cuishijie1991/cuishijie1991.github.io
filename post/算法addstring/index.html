算法——字符串相加

addString

题目

给定两个字符串形式的非负整数 num1 和num2 ，计算它们的和。

注意：

num1 和num2 的长度都小于 5100. num1 和num2 都只包含数字 0-9. num1 和num2 都不包含任何前导零。 你不能使用任何內建 BigInteger 库， 也不能直接将输入的字符串转换为整数形式。 leetcode真题（easy）

#解题思路和难点

  • 字符串相加，低位相加，有进位问题
  • char转换成数字
  • 结果缓存形式，用char[]还是StringBuilder。建议使用stringBuilder,做结果反转和转string都很方便。

#一种解法

 class Solution {
    public String addStrings(String num1, String num2) {
        StringBuilder sb = new StringBuilder();
        int len1 = num1.length();
        int len2 = num2.length();
        int resLen = Math.max(len1, len2);
        int over = 0;
        for (int i = 0; i < resLen; i++) {
            int n1 = i < len1 ? num1.charAt(len1 - i - 1) - '0' : 0;
            int n2 = i < len2 ? num2.charAt(len2 - i - 1) - '0' : 0;
            int total = n1 + n2 + over;
            int res = (total % 10);
            sb.append(res);
            over = total / 10;
        }
        //如果有进位，退出循环后需要进位
        if (over==1){
            sb.append(over);
        }
        //用stringBuilder的翻转字符串
        return sb.reverse().toString();
    }
}
