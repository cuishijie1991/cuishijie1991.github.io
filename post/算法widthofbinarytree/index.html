算法——最大二叉树宽度

widthOfBinaryTree

Posted by Tracy on Saturday, March 7, 2020

TOC

题目

给定一个二叉树，编写一个函数来获取这个树的最大宽度。树的宽度是所有层中的最大宽度。这个二叉树与满二叉树（full binary tree）结构相同，但一些节点为空。

每一层的宽度被定义为两个端点（该层最左和最右的非空节点，两端点间的null节点也计入长度）之间的长度。

示例 1:

输入:

       1
     /   \
    3     2
   / \     \  
  5   3     9

输出: 4 解释: 最大值出现在树的第 3 层，宽度为 4 (5,3,null,9)。 示例 2:

输入:

      1
     /  
    3    
   / \       
  5   3

输出: 2 解释: 最大值出现在树的第 3 层，宽度为 2 (5,3)。 示例 3:

输入:

      1
     / \
    3   2 
   /        
  5

输出: 2 解释: 最大值出现在树的第 2 层，宽度为 2 (3,2)。 示例 4:

输入:

      1
     / \
    3   2
   /     \  
  5       9 
 /         \
6           7

输出: 8 解释: 最大值出现在树的第 4 层，宽度为 8 (6,null,null,null,null,null,null,7)。 注意: 答案在32位有符号整数的表示范围内。

leetcode真题（medium）

#解题思路和难点

  • 求二叉树宽度，考虑BFS或DFS
  • 宽度是最两边节点的差，中间空节点也算，所以应该考虑记录数组下标运算。如果根节点中root节点是n，则左子节点是2n，右子节点是2n+1
  • 分层计算下一层的最大宽度，使用队列
  • 队列offer、peek、poll比较难记，可以用add、remove方法

#一种解法

    class Solution {
    public int widthOfBinaryTree(TreeNode root) {
        if (root == null) return 0;
        //队列保存节点
        Queue<TreeNode> queue = new LinkedList<>();
        queue.add(root);
        //队列链表保存最大宽度
        LinkedList<Integer> list = new LinkedList<>();
        list.add(1);
        int maxWidth = 1;
        //开启while循环逐层遍历
        while (!queue.isEmpty()) {
            int size = queue.size();
            //while中每次遍历一层
            for (int i = 0; i < size; i++) {
                //取出头部node和对应index
                TreeNode node = queue.remove();
                int index = list.remove();
                if (node.left != null) {
                    queue.add(node.left);
                    list.add(index * 2);
                }
                if (node.right != null) {
                    queue.add(node.right);
                    list.add(index * 2 + 1);
                }
            }
            //计算最大宽度，当前子层的最大宽度
            if(list.size()>0) {
            maxWidth = Math.max(maxWidth, list.get(list.size() - 1) - list.get(0) + 1);
            }
        }
        return maxWidth;
    }
}

}
